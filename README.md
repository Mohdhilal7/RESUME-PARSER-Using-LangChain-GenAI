# 📄 AI-Powered Resume Parser using LangChain

## 🚀 Overview
In today’s competitive recruitment landscape, manual resume screening is a time-consuming and error-prone process. This project provides an intelligent, automated solution: an **AI-powered Resume Parser** built using **LangChain** and Generative AI. 

By leveraging advanced Large Language Models (LLMs), this system seamlessly extracts key candidate information—such as personal details, education, skills, and work experience—from unstructured documents (PDFs, DOCX) and converts it into a highly accurate, machine-readable format. 

## ✨ Key Features
* **Multi-Format Document Extraction:** Reliably extracts text from unstructured `.pdf` and `.docx` files.
* **LLM-Powered Structuring:** Uses LangChain and generative AI models (e.g., GPT) to interpret, categorize, and structure resume data.
* **Intelligent Prompt Engineering:** Employs Prompt Templates and LLMChains for context-aware extraction, ensuring accurate identification of components regardless of inconsistent resume layouts.
* **Interactive Web Interface:** Features a user-friendly frontend built with Streamlit, allowing recruiters to upload resumes and visualize the structured output in real time.
* **Scalable & Efficient:** Significantly reduces manual profiling effort, minimizing information loss through iterative refinement.

## 🛠️ Tech Stack
* **Core Language:** Python
* **LLM Framework:** LangChain
* **Generative AI:** GPT-based Models (e.g., OpenAI API)
* **Text Extraction:** `PyPDF2`, `docx2txt`
* **Web UI:** Streamlit

## 💡 Future Scope & Extensions
This foundational parsing system can easily be extended into a comprehensive HR automation suite, with potential future features including:
* Automated Candidate-to-Job matching.
